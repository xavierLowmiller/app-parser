# app-parser

Parses metadata from `.apk` and `.ipa` files

## Features

| Android | iOS   | Feature                   |
| ------- | ----- | ------------------------- |
| - [ ]   | - [x] | Display Name              |
| - [ ]   | - [x] | Version                   |
| - [ ]   | - [x] | Bundle/Package Identifier |
| - [ ]   | - [x] | OS                        |
| - [ ]   | - [ ] | Minimum OS Version        |
| - [ ]   | - [ ] | Icons                     |
