# app-parser

Parses metadata from `.apk` and `.ipa` files

## Features

| Android | iOS                | Feature                   |
| ------- | ------------------ | ------------------------- |
| :x:     | :white_check_mark: | Display Name              |
| :x:     | :white_check_mark: | Version                   |
| :x:     | :white_check_mark: | Bundle/Package Identifier |
| :x:     | :white_check_mark: | OS                        |
| :x:     | :x:                | Minimum OS Version        |
| :x:     | :x:                | Icons                     |
